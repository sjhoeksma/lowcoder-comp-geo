Using the Lowcoder GEO plugin

This plugin enables the usage of Openlayers within lowcoder. To use this plugin.

  1. Open your lowcoder environment >> go to the app >> Select insert tab click on extentions.

  1. write the name of the npm package, for the our extentions will be : lowcoder-comp-geo or the full url of the package https://www.npmjs.com/package/lowcoder-comp-geo

  1. Drag and drop the geo component, Start using the plugin adding different type of layers like :
    • GeoJSON
    • Cloud Optimized GeoTIFF (COG)
    • PMTiles (Raster and Vector)
    • XYZ
    • WMS
    • WFS
    • Mapbox Vector Tiles ({z}/{x}/{y}.pbf)
    • Mapbox Style GL Json
    • ArcGIS FeatureServer (with Esri Style)
    • ArcGIS MapServer
    • ArcGIS ImageServer
    • ArcGIS TileServer
    • ArcGIS VectorTileServer (with Esri Style)
